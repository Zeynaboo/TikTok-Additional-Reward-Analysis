# Methodology

## Collection

- TikTok Studio mobile for reward metrics
- TikTok Studio desktop for performance, traffic, and audience metrics
- AutoHotkey for semi-automated collection

## Transformation

- Power Query cleanup
- Text normalization
- `Video_ID` generation
- Manual patch logic for special reward cases

## Modeling

- Star schema with engagement, traffic/SEO, and audience tables

## Limits

- Sample size remains moderate
- Some fields required manual validation
- Findings should be treated as exploratory
